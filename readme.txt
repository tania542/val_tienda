requerimientos para ejecutar el proyecto

1. instalamos xampp
2. copiamos la carpeta de "val_tienda" dentro de C:\xampp\htdocs
3. activamos el xampp y activamos el Apache y el MySQL
4. creamos la base de datos abrimos phpAdmin y creamos la base de datos llamada tienda_val
5. importamos el archivo base_datos/tienda_val.sql
6. abrimos el navegador y escribimos http://localhost/val_tienda
